# beamerthemeIllinois.sty

[![pres.pdf](./pres.png)](./pres.pdf)

A Beamer theme and presentation template built on the "metropolis" theme, tailored for the University of Illinois Urbana-Champaign

## Notable features

- Splash image as background of title slide

- Section navigation bar in the footline

- Total slide count excludes Appendix slides
